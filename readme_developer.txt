Запуск приложения "Ishod-w.exe" в режиме отладки осуществлялась через Spyder.
Необходимо открыть в Spyder файл main_ishod-w.pyw.

/* ******************************************************************************* */
/* ******************************************************************************* */

На данный момент исполняемый файл находится в папке "dist" под названием "main_ishod-w.exe".
Сборку exe файла осуществлялась с помощью библиотеки PyInstaller:
           > pip install PyInstaller
           > pyinstaller -F -w main_ishod-w.pyw
Исполняемый файл много весит и не кроссплатформенный с этим в будущем будем разбираться.

/* ******************************************************************************* */
/* ******************************************************************************* */

Во время разработки приложения использовались следующие команды.

1. Переход в папку ui_files для конвертации .ui-файл в Python-файл:
      > cd C:\Work_Xoma\GitHub\ui_files

2. Конвертация .ui-файлов в Python-файл с названиями ui_ishod_w.py, ui_dialogChangeExtensions.py:
      > pyuic5 -o ui_ishod_w.py ishod_w.ui
      > pyuic5 -o ui_dialogChangeExtensions.py dialogChangeExtensions.ui

3. Переход из папки ui_files для конвертации .ui-файл в Python-файл:
      > cd ..

4. Компилция resources.qrc в файл Python с именем resources_rc.py:
      > pyrcc5 -o resources_rc.py resources.qrc
